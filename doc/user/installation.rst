============
Installation
============

.. contents::
   :local:

You can install hop-SNalert-app via pip or from source.

To install with pip:

.. code:: bash

   pip install -U hop-SNalert-app

To install from source:

.. code:: bash

    tar -xzf hop-SNalert-app-x.y.z.tar.gz
    cd hop-SNalert-app-x.y.z
    python setup.py install
